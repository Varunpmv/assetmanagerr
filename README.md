# Asset Access Manager

A centralized system to manage employee access to various tools and assets across the organization.

## Features
- Consolidated view of all user access permissions.
- Tool-wise and employee-centric search/filtering.
- RACI matrix management for assets.
- Automated monthly access reviews.
- Role-based access control (Admin, Auditor, Reviewer, Approver, User).
- CSV import/export for employees and assets.

## Quick Start
1. Ensure Docker and Docker Compose are installed.
2. Run `docker-compose up -d --build`.
3. Access the frontend at `http://localhost:5173`.
4. Access the API at `http://localhost:5051`.
